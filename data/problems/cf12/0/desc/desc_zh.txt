# 题面
你在一家大公司的技术支持质量控制部门工作。你的工作是确保所有客户问题都得到解决。

今天，您需要检查客户与技术支持经理之间对话的副本。根据工作规则，客户的每条信息之后都必须有**条或几**条信息，这就是技术支持经理的回答。但是，有时客户提出的问题太快，以至于在客户提出一些新问题后，出现了一些经理对旧问题的答复。

由于隐私政策的原因，您无法看到信息的全文，只能看到信息的顺序，以及每条信息的类型：客户问题或技术支持经理的答复。**保证对话以客户问题开始**。

您必须确定该对话是否符合上述工作规则，或者是否违反了规则。

solution的主函数如下：  
```cpp
class Solution {
public:
    string solve(int &n, string &s) {
        // write your code here
    }
};
```
其中：  
- `n` 表示对话中的信息数量。
- `s` 是一个字符串，表示对话中的信息类型。`s[i]` 是 `c` 表示客户问题，`s[i]` 是 `m` 表示技术支持经理的回答。
- 函数应该返回一个字符串，如果对话符合工作规则，返回 "YES"，否则返回 "NO"。

# 示例 1  
- 输入：  
n = 3
s = "QAA"
- 输出：  
YES

# 约束条件  
- $1 \leq n \leq 100000$
- 时间限制：1000 ms  
- 内存限制：6400 KB
