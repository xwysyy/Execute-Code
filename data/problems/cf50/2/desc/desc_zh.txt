# 题面
给你一个由正整数组成的数组 $a_1, a_2, \ldots, a_n$ 。

你可以将数组中的一些元素涂成红色，但不能有两个相邻的红色元素（即对于 $1 \leq i \leq n-1$ 来说， $a_i$ 和 $a_{i+1}$ 中至少有一个元素不能是红色的）。

您的得分是红色元素的最大值加上红色元素的数量。请找出您能得到的最高分。

solution的主函数如下：  
```cpp
class Solution {
public:
    int solve(int &n, vector<int> &a) {   
        // write your code here
    }
};
```
其中：  
- 返回值为得到的最高分

# 示例 1：  
- 输入：  
n = 3
a = [5, 4, 5]
- 输出：  
7

# 约束条件：  
- $1 \leq n \leq 1000000$
- $1 \leq a[i] \leq 10^9$ 
- 时间限制：1000 ms  
- 内存限制：64 KB
