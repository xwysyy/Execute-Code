# 题面
给定一个由 $n$ 个元素组成的数组 $a$ ，打印任何至少出现三次的值，如果没有这样的值，则打印 -1。

solution的主函数如下：  
```cpp
class Solution {
public:
    int solve(int &n, vector<int> &a) {   
        // write your code here
    }
};
```
其中：  
- `n` 是一个整数，表示数组的长度。
- `a` 是一个整数数组，表示给定的数组。
- 返回值是一个整数，表示最小的至少出现三次的值，如果没有这样的值，则返回-1。

# 示例 1  
- 输入：  
n = 5
a = [1, 2, 3, 2, 2]
- 输出：  
2

# 约束条件  
- $1 \leq n \leq 100000$
- $1 \leq a[i] \leq n$
- 时间限制：1000 ms  
- 内存限制：6400 KB
