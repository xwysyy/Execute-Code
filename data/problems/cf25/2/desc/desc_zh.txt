# 题面
一个冬日的早晨，鲁道夫若有所思地看着窗外飘落的雪花。他很快就注意到雪花的形状具有某种对称性。鲁道夫像一个真正的数学家一样，想出了一个雪花的数学模型。

他将雪花定义为按照以下规则构建的无向图：

- 最初，图形只有一个顶点。
- 然后，向图中添加更多顶点。初始顶点通过边连接到 $k$ 个新顶点（ $k > 1$ ）。
- 每个只与一个其他顶点相连的顶点通过边与 $k$ 个新顶点相连。这一步至少要做**次**。

经过一些数学研究，鲁道夫意识到这种雪花可能没有任何顶点数。请帮助鲁道夫检查是否存在顶点为 $n$ 的雪花。

solution的主函数如下：  
```cpp
class Solution {
public:
    string solve(int &n) {   
        // write your code here
    }
};
```
其中：  
- 如果是顶点为 $n$ 的雪花，返回 "YES"，否则返回 "NO"。

# 示例 1：  
- 输入：  
n = 15
- 输出：  
"YES"

# 约束条件：  
- $1 \leq n \leq 1000000$
- 时间限制：1000 ms  
- 内存限制：64 KB
