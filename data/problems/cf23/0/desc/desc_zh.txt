# 题面
给你一个由 $n$ 个元素组成的二进制数组 $a$ ，二进制数组是一个只由 $0$ 个元素和 $1$ 个元素组成的数组。

空格是指仅由 $0$ 个元素组成的**连续**元素段。

你的任务是找出最长空白区域的长度。

solution的主函数如下：  
```cpp
class Solution {
public:
    int solve(int &n, vector<int> &a) {   
        // write your code here
    }
};
```
其中：  
- 返回值：返回最长空白区域的长度。

# 示例 1：  
- 输入：  
n = 5
a = [1, 0, 0, 1, 0]
- 输出：  
2

# 约束条件：  
- $1 \leq n \leq 1000000$  
- $0 \leq a_i \leq 1$
- 时间限制：1000 ms  
- 内存限制：6400 KB
